# moonlight-gtksourceview

Moonlight theme for [GtkSourceView](https://gitlab.gnome.org/GNOME/gtksourceview) based on the [Moonlight VS Code Theme](https://github.com/atomiks/moonlight-vscode-theme) with a few modifications for personal preference.

## Install

Download [moonlight.xml](moonlight.xml) to `~/.local/share/gtksourceview-5/styles` directory so that the text editor will recognize the style scheme.

```sh
mkdir -p ~/.local/share/gtksourceview-5/styles
wget -O ~/.local/share/gtksourceview-5/styles/moonlight.xml github.com/Arnesfield/moonlight-gtksourceview/raw/main/moonlight.xml
```

## Contributing

This style scheme is very much **incomplete and lacking** and only covers some basic colors for [Gnome Text Editor](https://apps.gnome.org/TextEditor/). Please feel free to contribute and issue a [pull request](https://github.com/Arnesfield/moonlight-gtksourceview/pulls) to include other colors and styling that are based on the [Moonlight VS Code Theme](https://github.com/atomiks/moonlight-vscode-theme).
